# meine_dict

meine_dict is a Flutter mobile app created for students who want to learn German vocabulary in a practical and engaging way. The app brings together a large vocabulary database, different learning approaches, and a simple interface so learners can study consistently.

## Overview

This project is designed to help German learners improve their vocabulary through a structured study experience. It includes a large collection of words and supports multiple ways to practice and revisit them.

## Features

- Learn German vocabulary through a mobile app
- Access more than 11,000 vocabulary entries from bundled datasets
- Practice vocabulary using different learning flows and study modes
- Save favorite words for quick review
- Add your own words to personalize the learning experience
- Built with Flutter for Android and other supported platforms

## Tech Stack

- Flutter
- Dart
- Provider
- SharedPreferences
- flutter_tts
- file_picker

## Project Structure

- lib/screens/ — app screens such as home, vocabulary list, practice, favorites, and details
- lib/services/ — storage, TTS, and repository logic
- lib/models/ — word and progress models
- lib/providers/ — app state management
- lib/src/ — vocabulary data files

## Getting Started

### Prerequisites

- Flutter SDK installed
- Android Studio or another compatible editor
- An Android device or emulator

### Install dependencies

```bash
flutter pub get
```

### Run the app

```bash
flutter run
```

### Build an APK

```bash
flutter build apk --debug
```
